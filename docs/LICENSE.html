License for pictures used in articles are usually included within the article itself. If not, look below.

Theme based on nandomoreirame’s “simplest” theme.

The code for estimating and displaying the time to read an article comes from atekihcan.github.io and github.com/renyuanz/leonids.

The mastodon widget on the front page is from mastodon-timeline-widget.

The tags page has been possible thanks to this blog article.

The main font ‘Open Sans’ is Apache-licensed.

jekyll-plantuml is used to generate inline PlantUML diagrams. It has been slightly modified to fit my needs (look at _plugins/jekyll-plantuml.rb).

screenfull.js polyfill is used to enhance compatibility with the fullscreen DOM API (displaying PlantUML diagrams at full size).
